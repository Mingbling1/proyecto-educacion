# Simulador del Sistema Endocrino

Un modelo anatómico interactivo de escala realista para aprender sobre el sistema endocrino, sus glándulas, hormonas y su conexión con la vida diaria y enfermedades en el contexto peruano.

## Tecnologías
* HTML5, CSS3, JavaScript (Vanilla)
* Vite (Entorno de desarrollo)

## Cómo ejecutar localmente
1. Clona el repositorio
2. Ejecuta `npm install`
3. Ejecuta `npm run dev`
4. Abre `http://localhost:3000` en tu navegador.
